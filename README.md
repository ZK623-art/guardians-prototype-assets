## About This Repository

This repository contains visual assets for the **Guardians of the Shining Light** prototype - an interactive educational game addressing youth tobacco addiction in South Asian communities.

**Project Lead:** Dr. Zainab Kidwai (University of York)  
**Creative Director:** Daniel Bacchus  
**Status:** Pre-production prototype for funding applications
